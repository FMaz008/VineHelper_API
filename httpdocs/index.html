Vine Helper is a browser extension to improve the Vine Experience

This extension allow users of Amazon Vine to:
- Hide items
- Know which items are new using the bookmark feature
- Dark mode
- Fix the infinite loading wheel issue
- Identifies import fees in some countries
- Apply Thorvarium stylesheets
